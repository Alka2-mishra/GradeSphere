# GradeSphere 🎓
### AI-Powered Student Performance Analytics & Smart Learning Management System

GradeSphere is a full-stack AI-powered educational analytics and classroom management platform designed to improve student performance through intelligent monitoring, attendance tracking, quizzes, assessments, analytics, and real-time communication between students and teachers.

The platform provides separate dashboards for students and teachers, helping institutions digitize academic management and improve learning outcomes using data-driven insights and AI-based analytics.

---

# 🌟 Features

## 👨‍🏫 Teacher Features

- Teacher Authentication & Authorization
- Create & Manage Classrooms
- Upload Chapters, Notes & Study Materials
- Create Quizzes & Assessments
- Mark Attendance
- Publish Notices & Announcements
- Evaluate Assignments
- Analyze Student Performance
- Track Weak Students
- Real-Time Student Communication
- AI-Based Academic Insights
- Performance Analytics Dashboard
